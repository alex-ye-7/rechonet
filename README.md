# rechonet

A from-scratch reimplementation of [EchoNet-Dynamic](https://github.com/echonet/dynamic) (Ouyang et al., *Nature* 2020) — video-based deep learning for beat-to-beat
assessment of cardiac function. 

Built for my own learning; structure and core logic follow the original closely. Addition of key frames (systolic and diastolic) model under utils.
